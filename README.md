# 🎬 UIKit Animations Example
A UIKit project covering the core animation APIs — UIView animations, Core Animation layers, and spring effects.

---

## 🤔 What this is
This project demonstrates how to animate views in UIKit using `UIView.animate`, `UIView.transition`, keyframe animations, and CALayer-based Core Animation. It covers fade, slide, scale, and spring animations with different timing curves. A practical reference for anyone adding motion to a UIKit app.

## ✅ Why you'd use it
- **UIView animation APIs** — Covers `UIView.animate(withDuration:)`, keyframe animations, and spring damping
- **Core Animation** — Shows how to use CABasicAnimation and CAKeyframeAnimation for layer-level effects
- **Timing and curves** — Demonstrates ease-in, ease-out, spring, and custom cubic Bezier timing functions

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/UIKitAnimationsExample.git
cd UIKitAnimationsExample
```

### 2. Open in Xcode
Double-click `UIKitAnimationsExample.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- All animations run fine in the Simulator.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
